# Nexora AI 🚀

Modern AI-powered chatbot application built using React, Node.js, Express, MongoDB, and Groq API.

# 🌐 Live Demo

Frontend: [Open Nexora AI](https://nexoraai-bxs2.onrender.com)

Backend API: [Backend Server](https://nexora-ai-e15c.onrender.com)

---
Nexora AI is a modern AI-powered chatbot web application built using React, Node.js, Express, MongoDB, and Groq API.

Users can:
- Create new AI chats
- View chat history
- Delete conversations
- Get AI-generated responses instantly
- Enjoy a clean modern dark UI

---

- AI Chat Integration

# ⚙️ Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=8080

MONGODB_URI=your_mongodb_connection_string

GROQ_API_KEY=your_groq_api_key
```

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/nexora-ai.git
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 3️⃣ Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

# ▶️ Run Project

## Start Backend

```bash
cd backend
npm start
```

Backend runs on:

```bash
http://localhost:8080
```

---

## Start Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
